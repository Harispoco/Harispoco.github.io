Sales Order Analysis using Excel

An Excel-based data analysis project focusing on transaction ranking, yearly order trends, and grouped sales performance.

Project Overview

This project analyzes sales order data to extract meaningful insights related to top-value transactions, order volume by year, and total sales performance based on salesperson name initials. The analysis was conducted entirely using Microsoft Excel with advanced formulas and conditional logic.

Objectives

  • Identify the top two highest-value orders for each salesperson
  • Calculate the total number of orders for specific years
  • Analyze total sales based on salesperson initials and selected years

Step-by-Step Analysis Process

Step 1 – Order IDs for Top Two Orders per Salesperson

The first analysis identifies the Order IDs corresponding to the top two highest order amounts for each salesperson.

Excel Formulas

=IFERROR(
  INDEX($E$3:$E$801,
    MATCH(
      LARGE((I4=$C$3:$C$801)*$F$3:$F$801,ROW(A1)),
      $F$3:$F$801,0) ),"-")

Explanation (Step by Step)

(I4 = $C$3:$C$801)

→ Filters the dataset to only include records that match the selected salesperson.

* $F$3:$F$801

→ Multiplies the filtered results with the order amounts, keeping only the relevant sales values for that salesperson.

LARGE(..., ROW(A1))

→ Extracts the largest and second largest order values dynamically. The ROW function allows ranking (1st largest, 2nd largest).

MATCH(...)

→ Identifies the exact position of the selected order amounts within the dataset.

INDEX(...)

→ Returns the corresponding Order ID based on the matched position.

IFERROR()

→ Prevents formula errors by returning a placeholder value when fewer than two orders exist for a salesperson.

Outcome

Successfully identified the top two highest-value transactions for each salesperson, enabling quick recognition of high-impact sales.

Step 2 – Total Number of Orders by Year

The second analysis calculates the total number of orders for specific years based on the order date.

Excel Formulas

=SUMPRODUCT(--(YEAR(Order_Date)=H12))

Explanation (Step by Step)

YEAR(Order_Date)

→ Extracts the year from each order date

=H12

→ Filters orders by the selected year.

SUMPRODUCT(--())

→ Converts TRUE/FALSE results into numeric counts.

Outcome

Generated a clear comparison of order volume across different years, highlighting fluctuations in sales activity over time.

Step 3 – Total Sales by Salesperson Initial and Year

The final analysis calculates total sales amounts for salespeople whose names start with a specific letter in selected years.

Excel Formulas

=SUMPRODUCT(
  --(LEFT(Salesperson,1)=$H20),
  --(YEAR(Order_Date)=I$19),
  Order_Amount)

Explanation (Step by Step)

LEFT(Salesperson,1)

→ Extracts the first letter of each salesperson’s name

=H20

→ Filters by the selected initial (e.g., C, L, S).

YEAR(Order_Date)=I$19

→ Filters by year.

SUMPRODUCT()

→ Sums the order amounts that meet all conditions.

Outcome

Enabled grouped performance analysis based on salesperson initials and yearly trends.

Results & Insights

Results

Top Two Orders per Salesperson

The analysis successfully identified the top two highest-value orders for each salesperson. Salespeople such as Fuller, Davolio, Dodsworth, Leverling, Buchanan, Callahan, and Suyama recorded two significant high-value orders. Meanwhile, King and Peacock did not have qualifying top-ranked orders in the dataset, indicating either lower transaction values or limited order activity during the observed period. This ranking provides a clear comparison of individual sales impact based on transaction value.

Number of Orders by Year

The total number of orders per year shows a noticeable fluctuation:

  1. 2004 → 399 orders (Highest)
  2. 2005 → 257 orders
  3. 2003 → 143 orders (Lowest)
There was a significant increase from 2003 to 2004, followed by a decline in 2005.

Total Sales by Initial Letter of Salesperson

Sales performance grouped by the first letter of salesperson names shows the following:

1. 2004

C → $116,781.21

L → $235,384.63 (Highest)

S → $40,826.37

2. 2005

C → $88,821.59

L → $130,416.25 (Highest)

S → $17,181.58

3. 2003

C → $34,392.86

L → $69,169.07 (Highest)

S → $14,519.68

Across all years, salespeople whose names start with "L" consistently generated the highest total sales.

Insight

1. Peak Operational Year: 2004

2004 recorded the highest number of transactions (399 orders), indicating strong operational activity and market demand during that year.

2. Sales Contribution Concentration

Sales performance is heavily concentrated among specific individuals and name groups. The “L” group dominates total sales in all observed years.

3. Performance Disparity

The gap between the highest and lowest contributing groups is significant. For example, in 2004, the "L" group generated more than 5x the revenue of the "S" group.

4. Strategic Implication
  • Sales performance evaluation
  • Incentive program design
  • Targeted coaching for lower-performing groups
  • Resource allocation planning
5. Data-Driven Monitoring

The use of advanced Excel formulas such as Array Formulas and SUMPRODUCT enables scalable performance tracking for future datasets without manual filtering.

Business Value

This analysis helps management quickly identify top-performing sales transactions, monitor yearly trends, and evaluate grouped sales performance. The approach reduces manual reporting and supports data-driven decision making.

Tools & Skills Used

  • Microsoft Excel
  • Array formulas & SUMPRODUCT
  • Date and text-based data filtering
  • Analytical problem-solving
